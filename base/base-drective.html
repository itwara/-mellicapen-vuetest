vue常见指令：

v-if
v-show
v-show
v-else
{{value}}
v-on
v-bind
v-pre{{msg}}
{{ cloak }}

总结：

1.指令会被最终解析掉，不会在页面显示;指令后面的值会被当作表达式处理

2.v-text改变的是当前dom的textContent

3.v-html改变的是当前dom的innerHTML

4.v-if控制当前dom的删除与否，类似remove，而非detach

5.v-view控制当前dom的display

6.v-else与v-if|v-show配对

7.v-for遍历数组或对象，表达式的写法 item in/of items ,或者 (key,value) in/of items

8.v-on用在普通元素上时，只能监听原生 DOM 事件。用在自定义元素组件上时，也可以监听子组件触发的自定义事件。

9.v-bind动态地绑定一个或多个 attribute，或一个组件 prop 到表达式。

10.v-model指令在表单控件元素上创建双向数据绑定。

11.v-ref

12.v-el为 DOM 元素注册一个索引，方便通过所属实例的 $els 访问这个元素,通过this.$els.other

13.v-pre跳过编译这个元素和它的子元素

14.v-cloak当前的dom编译完才显示
